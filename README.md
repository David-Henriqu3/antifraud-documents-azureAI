# Análise de Documentos Anti-fraude com Azure AI 📂

Este projeto faz parte do [Bootcamp Microsoft Certification Challenge #5 - AI 102](https://www.dio.me/bootcamp/microsoft-certification-challenge-5-ai-102), com foco em fundamentos de IA aplicada, Azure AI Services e preparação para certificação.

## 📌 Objetivo do Projeto

Implementar uma solução capaz de analisar documentos armazenados no Azure Blob Storage utilizando o **Azure Document Intelligence**, auxiliando na identificação de inconsistências, validação de informações e apoio a processos antifraude.

## 🚀 Funcionalidades

- Upload e armazenamento de documentos no Azure Blob Storage  
- Análise automática de documentos com Azure Document Intelligence  
- Extração de informações estruturadas (texto, campos e dados relevantes)  
- Aplicação voltada para cenários de análise e prevenção a fraudes  

## 🧠 Tecnologias Utilizadas

- **Microsoft Azure AI Services**
- **Azure Document Intelligence**
- **Azure Blob Storage**
- **Python**
- **APIs REST do Azure**
